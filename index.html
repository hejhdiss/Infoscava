Key Features in v2.1.9

File Analysis

  • Metadata, hashes, text, hex, structured data (JSON)
  • Image (EXIF, GPS), Base64, entropy, byte histograms

User Interface

  • Tabbed design, file browser, drag-and-drop
  • Theme toggle, file watcher, keyboard shortcuts
  • Analysis export (JSON, TXT, HTML)
  • Search Tab with multi-match highlighting

Plugin System

  • Dynamic Plugin Loading (.infoscava files)
  • Plugin Management: Load, reload, delete plugins
  • Dynamic Plugin Tabs for custom output (HTML, JSON, text)
  • Plugin History log
